# NavAR
NavAR is a Location Based Augmented Reality Android App . It uses the [BeyondAR](https://github.com/BeyondAR/beyondar) framework for AR.

## Features
* Geocoding
* Reverse Geocoding
* Augmented Reality Point of Interest Browser
* Augmented Reality Navigation
